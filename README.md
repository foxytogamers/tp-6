##Trabajo Práctico N°6: Tabla Informativa con HTML y CSS

 Tema Seleccionado: Comparativa de Consolas de Novena Generación

Justificación de la Elección:

Se eligió la temática de "Comparativa de Consolas de Videojuegos de Novena Generación" porque es un tema con datos fácilmente estructurables y que permite el uso claro de los atributos avanzados de tabla solicitados:

Uso de colspan (Column Span): Se utilizó para crear el encabezado principal "Rendimiento Principal" que abarca las columnas de "CPU" y "GPU", agrupando lógicamente dos métricas relacionadas con la potencia de procesamiento.

Uso de rowspan (Row Span): Se implementó en la columna "Fabricante" (SONY y MICROSOFT) para agrupar visualmente los distintos modelos (con o sin disco, Series X y Series S) bajo una única celda de fabricante, mejorando la legibilidad y el orden de la tabla.

Además, el tema permite un diseño visual atractivo y el uso de colores alternados para diferenciar claramente las filas y mejorar la experiencia del usuario (legibilidad).
